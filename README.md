# Mavis-cares-quote-form
Mobile-first WhatsApp quote form for Mavis Cares & Nurses. Built with HTML, CSS, Vanilla JS. Captures leads via WhatsApp API.

**Live Demo:** https://lorraine-lolo.github.io/Mavis-cares-quote-form/

### Features
- Client selects service type + days per week + WhatsApp number
- Auto-opens WhatsApp with pre-filled quote request
- Tracks `form_start` and `whatsapp_click` events in Google Analytics 4

### Tech Stack
HTML | CSS | Vanilla JS | GitHub Pages | Google Analytics 4
